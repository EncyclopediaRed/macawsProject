Macaws Airplane Project Instructions

* Uses a JDBC Driver to connect to a MySQL Database

How to setup (Currently working variation)

1.0 Start XAMPP
2.0 Run 
3.0 Select "0" Menu Item in the program to load the SQL file
4.0 Connect to the database with Menu Option "0"
5.0 user and password = itp220, db = macaws (all hard coded into the program for ease of use)
6.0 Select options 1-X
7.0 Record results
8.0 Exit
